FAVICON NOTE
============

The website references favicon.jpg in all HTML files. 

To add a favicon:
1. Create or download a favicon image (recommended size: 32x32 or 64x64 pixels)
2. Save it as "favicon.jpg" in the root directory
3. Alternatively, you can use favicon.ico, favicon.png, or any other format
4. Update the <link rel="icon"> tag in all HTML files if using a different format

For best results, use a simple icon related to pools, water, or Dubai.
You can create one using online favicon generators or design tools.
